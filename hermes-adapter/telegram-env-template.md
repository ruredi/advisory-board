# Telegram Advisor Token Template

Do not commit real tokens here. This file documents what needs to be added to each Hermes profile `.env` file.

## Common value

```bash
TELEGRAM_ALLOWED_USERS=<ANDRAS_NUMERIC_TELEGRAM_USER_ID>
```

Get the numeric user ID from @userinfobot or @get_id_bot.

## Per-profile files

### ~/.hermes/profiles/jobs/.env

```bash
TELEGRAM_BOT_TOKEN=<STEVE_JOBS_BOT_TOKEN>
TELEGRAM_ALLOWED_USERS=<ANDRAS_NUMERIC_TELEGRAM_USER_ID>
```
